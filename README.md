# CS2 Case Lab

Статический фан-симулятор открытия кейсов CS2 для GitHub Pages.

## Что есть

- внутренняя валюта LabCoins;
- разные кейсы с разным пулом предметов;
- анимированное открытие кейса;
- локальный инвентарь через `localStorage`;
- продажа предметов;
- апгрейд предметов;
- бонусное колесо;
- рекламный просмотр на 10 секунд с наградой;
- блок проектов для саморекламы.

## Как запустить на GitHub Pages

1. Создай репозиторий на GitHub.
2. Загрузи `index.html`, `style.css`, `script.js`, `README.md` в корень репозитория.
3. Открой `Settings → Pages`.
4. В `Build and deployment` выбери `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. Сохрани и дождись ссылки GitHub Pages.

## Как заменить рекламные проекты

Открой `script.js` и найди массив `projects`. Замени `title`, `desc`, `url` на свои ссылки.

## Важно

Проект является фан-симулятором. В нем нет реальных платежей, вывода предметов, Steam-авторизации или азартной игры на деньги.
