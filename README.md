# CS2 Case Lab v31.4

Статический фан-симулятор открытия кейсов для GitHub Pages.

## Что изменено в v31.4

- Переработана ценовая модель для наклеек, патчей, сувенирных наборов, viewer pass / major pass, ножей, перчаток и других редких предметов.
- Убрано отображение старой смешанной валюты: цены и баланс теперь можно переключать в шапке между рублями, долларами, евро и LC.
- LC оставлен как отдельная игровая единица: `1 LC = $10`. Расчёты внутри save хранятся в рублях, переключатель меняет только отображение.
- Реалистичные шансы, реальные изображения и навигация по сериям из v31.3 сохранены.
- Сборка чистая: без базы данных, backend и `.bat` файлов.
