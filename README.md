# CS2 Case Lab v5

Статический фан-симулятор открытия кейсов CS2 для GitHub Pages.

## Что исправлено в v5

- Новый ключ сохранения `cs2_case_lab_v5_state`, поэтому старые сломанные сохранения из предыдущих сборок не ломают баланс.
- Баланс принудительно приводится к числу перед каждым списанием и начислением.
- Кнопки работают через единый обработчик событий, поэтому модальные окна, кейсы, колесо, продажа, апгрейд и контракты не отваливаются после перерисовки DOM.
- `start-local.bat` больше не пытается сам открыть браузер, чтобы не появлялась ошибка Windows `Application not found`.
- В HTML подключены `app.js?v=5.1.0` и `styles.css?v=5.1.0`, чтобы GitHub Pages/браузер не оставлял старый кеш.

## Локальный запуск

1. Распакуй ZIP.
2. Открой папку `cs2-case-lab-v5`.
3. Запусти `start-local.bat`.
4. Вручную открой в браузере:

```text
http://localhost:8000/index.html
```

Не запускай `python -m http.server 8000` из `C:\Windows\System32`.

## GitHub Pages

Загрузи все файлы из папки проекта в репозиторий. В настройках GitHub Pages выбери ветку `main` и папку `/root`.

Если после замены файлов сайт ведёт себя как старая версия, нажми Ctrl+F5 или очисти кеш сайта.

## Замена рекламных проектов

В `app.js` найди функцию `projectCards()` и замени `#` на свои ссылки.

## Важно

Это фан-проект. В нём нет реальных платежей, депозитов, Steam-логина, вывода предметов и азартных ставок.
