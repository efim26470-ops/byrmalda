# CS2 Case Lab v2

Многостраничный фан-симулятор открытия кейсов CS2 для GitHub Pages.

## Что реализовано

- несколько HTML-страниц: главная, кейсы, инвентарь, апгрейд, контракты, бонусы, баттлы, реклама, профиль;
- внутриигровая валюта LabCoins;
- 10 кейсов с разными ценами и пулами предметов;
- 40 процедурно отрисованных предметов: пистолеты, SMG, винтовки, AWP/SSG, дробовики, ножи и перчатки;
- анимированная рулетка открытия кейса;
- модальное окно после дропа: продать, оставить, отправить в апгрейд, отправить в контракт, открыть ещё;
- рабочий инвентарь с фильтрами, сортировкой и сохранением через localStorage;
- апгрейд предметов с расчетом шанса;
- контракты 3–10 предметов;
- локальные case battle против ботов;
- бонусное колесо, daily reward, промокоды;
- встроенный рекламный просмотр на 10 секунд с начислением монет;
- фейковая live-лента дропов;
- адаптивная верстка для телефона и ПК.

## Важно

Это фан-симулятор. В проекте нет реальных платежей, вывода скинов, Steam-логина, азартной игры на деньги и связи с Valve / Counter-Strike / Steam.

## Как заменить рекламу на свои проекты

Открой `app.js` и найди массив:

```js
const projects = [
  {title:'Портфолио Ефима', desc:'...', url:'#', badge:'WEB'},
  ...
];
```

В поле `url` вставь ссылки на свои GitHub Pages, YouTube, Яндекс.Диск и т.д.

## Как опубликовать на GitHub Pages

1. Создай репозиторий на GitHub.
2. Загрузи все файлы из папки проекта в корень репозитория.
3. Открой `Settings` → `Pages`.
4. В `Build and deployment` выбери `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. Сохрани и дождись ссылки GitHub Pages.
