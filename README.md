# CS2 Case Lab

Статический HTML-проект для GitHub Pages: кейсы, инвентарь, апгрейд, контракты, battle, колесо, реклама, промокоды и внутренняя валюта ₽LC.

## Установка на GitHub Pages

1. Загрузи содержимое папки проекта в репозиторий.
2. Включи GitHub Pages для ветки `main`.
3. После обновления сайта сделай полную перезагрузку страницы.

## Локальный запуск

Открой `index.html` через локальный сервер или запусти `start-local.bat`.

## Сохранение

Прогресс хранится в браузере: `localStorage`, `IndexedDB`, `sessionStorage` и резервный fallback.
