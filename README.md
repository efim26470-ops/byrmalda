# CS2 Case Lab v8

Фан-симулятор открытия кейсов CS2 для GitHub Pages. Без платежей, Steam-логина, вывода предметов и реального гемблинга.

## Что исправлено в v8

- Новый стабильный ключ сохранения: `cs2_case_lab_state`, больше не меняется от версии к версии.
- Автобэкап сохранения: `cs2_case_lab_state_backup`.
- Импорт старых сохранений v2-v7.
- Отключён старый Service Worker/cache, который мог отдавать старый `app.js` и ломать баланс/инвентарь.
- Баланс списывается до старта кейса/баттла, предмет добавляется в инвентарь после результата.
- Инвентарь перечитывает сохранение при открытии страницы.
- Case Battle переписан: нормальная арена, режимы 1v1, 1v1v1, 2v2, анимация рулеток, итог и начисление пула при победе.
- Добавлены расширенные пулы: официальные кейсы, коллекции, Armory/collection-пулы, quality-кейсы, наклейки турниров, агенты, брелоки, нашивки.

## Ограничения

Живые Steam-цены на чистом GitHub Pages без backend/API-ключа надёжно не получить из-за CORS, лимитов и отсутствия сервера. В проекте используется Steam-like оценка LC, основанная на редкости, названии и известных дорогих паттернах.
