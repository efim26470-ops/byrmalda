# CS2 Case Lab v31.6

Статический фан-симулятор открытия кейсов CS2 для GitHub Pages.

## Что изменено в v31.6

- Исправлена пропавшая картинка `Sticker | Team Dignitas (Holo) | Katowice 2014` в финальном Dragon Hoard Case.
- Добавлена отдельная вкладка `Season pass` (`seasonal-pass.html`).
- Seasonal battle-pass активируется за `$20`, имеет 30 уровней и доступен 30 дней в текущем сохранении, затем вкладка скрывается из меню.
- Сезонные задания проще основного Battle-pass: открытия кейсов, быстрые открытия, продажи, колесо, апгрейды, контракты, sticker-разделы.
- Награды сезонного pass скромнее: баланс, предметы, сезонные промокоды, Season Drop Case и финальный Season Final Case.
- Основной Battle-pass на 100 уровней, валюты ₽ / $ / € / LC, реальные картинки, цены и навигация по сериям сохранены.
- Сборка чистая: без базы данных, backend и `.bat` файлов.

## Как залить на GitHub Pages

1. Распакуйте архив.
2. Загрузите содержимое папки `cs2-case-lab-v31-6` в корень репозитория GitHub Pages.
3. После обновления откройте сайт с параметром `?clear=v316` или очистите кэш браузера.

Прогресс пользователя хранится только в браузере через localStorage/IndexedDB.
