# CS2 Case Lab v23

Статический фан-симулятор для GitHub Pages.

В этой версии исправлен главный экран: вместо одной обрезанной картинки используется адаптивная витрина из отдельных карточек скинов.


## v25
- Главная витрина использует жестко заданные реальные URL изображений AK-47 Redline, AWP Hyper Beast и Karambit Doppler.
- Мобильные кнопки обрабатываются напрямую через click/touchend/pointerup.
