# CS2 Case Lab v31.1 Lite

Лёгкая версия для GitHub Pages.

Что оптимизировано:
- удалены тяжёлые iOS splash-изображения и неиспользуемые hero-ассеты;
- удалены лишние размеры иконок, оставлены PWA/iOS/Windows-иконки, которые реально подключены;
- отключена автоматическая загрузка огромного онлайн-каталога JSON; сайт сразу работает на встроенном каталоге;
- онлайн-каталог можно временно включить вручную: `index.html?online=1`;
- чистка старого Service Worker/кэша выполняется один раз или через `?clear=v31lite`, а не при каждом открытии;
- уменьшено число DOM-элементов live-ленты и частота её перерисовки.

Как залить:
1. Загрузите содержимое этой папки в корень репозитория GitHub Pages.
2. После публикации откройте сайт один раз с `?clear=v31lite`.
3. Дальше используйте обычную ссылку без параметров.

Без database, bat, backend и лишних сборочных файлов.
