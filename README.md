# CS2 Case Lab v31

Статический фан-симулятор открытия кейсов для GitHub Pages.

## Что изменено в v31

- Обновлена экономика дропа: сверхдорогие предметы теперь имеют сильное понижение веса относительно цены кейса.
- Обычные оружейные кейсы ориентируются на реальные пропорции редкостей CS2: Mil-Spec, Restricted, Classified, Covert и Rare Special.
- Цены предметов переведены в рублёвую логику `₽LC`, обновлена таблица популярных скинов и базовые коэффициенты для ножей, перчаток, наклеек, агентов, брелоков и патчей.
- Убраны тестовые кнопки начисления монет из интерфейса.
- Убраны лишние `.bat` и offline/download ZIP-файлы. Сборка чистая и статическая.
- PWA/иконки/iOS-установка сохранены.

## Как залить на GitHub Pages

1. Распакуйте архив.
2. Загрузите содержимое папки `cs2-case-lab-v31` в корень репозитория GitHub Pages.
3. После обновления откройте сайт с параметром `?clear=v31` или очистите кэш браузера.

Никакая база данных, сервер или `.bat` файл не нужны. Прогресс пользователя хранится только в браузере через localStorage/IndexedDB.
