# CS2 Case Lab v3

Статический многостраничный фан-симулятор открытия кейсов для GitHub Pages.

## Что есть

- многостраничная структура: `index`, `cases`, `inventory`, `upgrade`, `contracts`, `wheel`, `battle`, `ads`, `profile`;
- локальная валюта LabCoins;
- 12 кейсов и 60 предметов;
- локальные PNG-рендеры предметов и кейсов в папке `assets`;
- рабочее открытие кейсов с рулеткой;
- списание баланса при открытии;
- начисление предмета в инвентарь после остановки рулетки;
- модальное окно после дропа: продать, оставить, отправить в апгрейд, отправить в контракт, открыть ещё;
- рабочий инвентарь с продажей, фильтрами и сортировкой;
- апгрейд предметов;
- контракты 3–10 предметов;
- бонусное колесо, daily reward и промокоды;
- fake case battle против ботов;
- fake live feed;
- рекламная страница твоих проектов с 10-секундным просмотром и начислением монет.

## Как запустить локально

Открой `index.html` в браузере или запусти локальный сервер:

```bash
python -m http.server 8000
```

Потом открой:

```text
http://localhost:8000
```
