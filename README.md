# CS2 Case Lab v31.3

Чистая версия для GitHub Pages.

Что внутри:
- база и внешний вид восстановлены от версии v31;
- сохранены реалистичные шансы и рублёвая экономика v31;
- на странице кейсов добавлены кнопки разделов: классические, оружейные, лимитированная серия, качество, ножи/перчатки, наклейки, агенты и другое;
- добавлены плавающие кнопки «Вверх» и «Вниз»;
- включён онлайн-каталог CSGO-API, чтобы кейсы и скины подтягивали реальные изображения;
- SVG-заглушки больше не подставляются в карточки кейсов/дропа/рулетки: если картинка не загрузилась, пробуется второй CDN-источник, затем изображение скрывается;
- без database, backend и bat-файлов.

Загрузка на GitHub Pages: скопируйте содержимое этой папки в корень репозитория и откройте `cases.html?clear=v313` для сброса старого кэша.
