# CS2 Case Lab v9

Статический фан-симулятор открытия кейсов CS2 для GitHub Pages.

## Что исправлено в v9

- Переписано сохранение: теперь используется компактный стабильный ключ `cs2_case_lab_save`.
- Старые сломанные ключи v3–v8 автоматически удаляются, чтобы не забивать quota localStorage.
- Добавлен sessionStorage backup: если браузер временно не даёт писать в localStorage, прогресс держится хотя бы в текущей вкладке.
- Инвентарь сохраняется в компактном виде и перечитывается при переходах между страницами.
- Кейсы получили тематические обложки с превью реальных предметов из пула: зелёные/красные/синие quality-кейсы, турниры, агенты, брелоки, нашивки, ножи и перчатки.
- Service Worker не кеширует старые версии, чтобы GitHub Pages не подсовывал старый JS.

## Как залить на GitHub Pages

1. Распакуй ZIP.
2. Загрузи **содержимое папки** `cs2-case-lab-v9` в корень репозитория.
3. В GitHub: Settings → Pages → Deploy from branch.
4. После деплоя открой сайт и нажми `Ctrl + F5`.
5. При старых глюках: Профиль → Сбросить прогресс.

## Важно

Проект не связан с Valve/Steam. Нет реальных платежей, депозитов, вывода предметов и ставок. Все LC, предметы и цены используются только для фан-симуляции.
