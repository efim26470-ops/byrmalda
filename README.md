# CS2 Case Lab v6

Многостраничный статический фан-симулятор открытия кейсов CS2 для GitHub Pages.

## Что добавлено в v6

- Исправлено отображение содержимого кейса: компактная сетка без горизонтального развала.
- Добавлена центральная стрелка/метка DROP у рулетки, чтобы было понятно, какой предмет выпадает.
- Расширен ассортимент: официальные кейсы, коллекции CS2 и отдельные пулы по качествам: Mil-Spec, Restricted, Classified, Covert, ножи, перчатки, ножи + перчатки.
- Колесо бонусов: лимит 1 прокрутка в 2 часа.
- Реклама: обязательное модальное окно на 10 секунд, закрытие до конца таймера заблокировано.
- Лимит рекламы: 10 просмотров в сутки на браузер/устройство через localStorage. Настоящий лимит по IP невозможен без бэкенда.
- Апгрейд получил анимацию рулетки WIN/LOSE.
- Добавлена страница `install.html`: кнопка установки PWA для Windows, инструкция для iOS и ZIP-сборка.
- Добавлены PNG-иконки, manifest и service worker для PWA.
- Все подключения обновлены до `?v=6.0.0`, чтобы GitHub Pages не держал старый кеш.
