# CS2 Case Lab v30

Статический фан-симулятор для GitHub Pages. В v30 исправлен запуск вкладок: страницы рендерятся сразу на встроенном каталоге, внешний каталог подгружается фоном. Главная витрина использует жестко заданные реальные URL изображений AK-47 Redline, AWP Hyper Beast и Karambit Doppler.

Загружайте содержимое папки в корень репозитория GitHub Pages. После обновления откройте `index.html?clear=v30`.
